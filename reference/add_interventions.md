# Add all interventions

Add all interventions

## Usage

``` r
add_interventions(p, interventions, species)
```

## Arguments

- p:

  parameter list

- interventions:

  site intervention inputs

- species:

  Can be falciparum: "pf" or vivax: "pv", for vivax SMC, RTSS, R21 and
  PMC are not implemented

## Value

modified parameter list
