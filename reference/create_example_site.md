# Create a skeleton example site

Create a skeleton example site

## Usage

``` r
create_example_site(itn_use = c(0.1, 0.2, 0.5))
```

## Arguments

- itn_use:

  Numeric vector. ITN use values to pass to
  \`create_example_interventions()\` (default: c(0.1, 0.2, 0.5)).

## Value

Skeleton example single site
